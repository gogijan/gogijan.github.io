# Политика конфиденциальности

Дата обновления: 4 мая 2026 г.

`Помощник кабинета ГИС ЖКХ` - неофициальное расширение браузера для работы на `dom.gosuslugi.ru` и его поддоменах.

Расширение не является официальным продуктом ГИС ЖКХ, Госуслуг, ЕСИА или Минстроя России и не связано с владельцами этих сервисов.

## Какие данные обрабатывает расширение

Расширение работает локально в браузере пользователя и может обрабатывать:

- URL текущей вкладки, чтобы определить, открыта ли страница `dom.gosuslugi.ru` или его поддомена;
- состояние страниц личного кабинета ГИС ЖКХ, нужное для включенных пользователем автоматических действий;
- cookie `route_sso` домена `dom.gosuslugi.ru`, чтобы удалить ее при переходе к повторному выбору организации;
- краткое название текущей организации, если официальный сайт возвращает его в ответе состояния пользователя;
- настройки расширения;
- локальный журнал последних технических действий расширения.

Расширение не собирает логины, пароли, коды подтверждения, токены ЕСИА, платежные данные и содержимое личных сообщений.

## Как используются данные

Данные используются только для функций расширения:

- переход к повторному выбору организации в ГИС ЖКХ;
- показ состояния в popup;
- автоматическое подтверждение входа в организацию, если эта настройка включена;
- автоматическое закрытие информационных сообщений, если эта настройка включена;
- автоматическое продолжение сессии по запросу сайта, если эта настройка включена;
- хранение локальных настроек и локального журнала последних действий.

## Передача данных

Расширение не передает данные разработчику, рекламным сетям, аналитическим сервисам или другим третьим лицам.

Расширение взаимодействует только со страницами и HTTPS-адресами `dom.gosuslugi.ru` и его поддоменов, где оно установлено пользователем и где нужны его функции.

## Хранение данных

Настройки и журнал последних действий хранятся локально в браузере через `chrome.storage`. Журнал содержит только технические записи без cookie, логинов, паролей, токенов и кодов подтверждения.

Пользователь может очистить журнал на странице настроек расширения или удалить все локальные данные, удалив расширение из браузера.

## Реклама и продажа данных

Расширение не показывает рекламу, не продает данные и не использует данные для персонализированной рекламы, скоринга, перепродажи или иных целей, не связанных с основной функцией расширения.

## Ограниченное использование

Использование информации расширением соответствует Chrome Web Store User Data Policy, включая Limited Use requirements: данные используются только для пользовательских функций расширения, не передаются третьим лицам, не используются для рекламы и не просматриваются людьми.

## Контакты

По вопросам конфиденциальности и работы расширения используйте контакт разработчика, указанный в карточке Chrome Web Store.
